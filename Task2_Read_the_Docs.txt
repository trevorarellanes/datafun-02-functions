2. What does math.ceil() do?  How many arguments does it take? What does it return? 
    Math.ceil rounds a number up to the nearest integer, if necessary, and returns the results
    It takes 1 arguments
    It returns the result which can be an int or float type argument

    math.fabs() returns the absolute value 
    It takes 1 argument
